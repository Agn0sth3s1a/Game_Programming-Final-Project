# Game_Programming-Final-Project
This is the final project for my Game Programming class, it is a little game that I've called Skullington. It's a little "platformer" where you don't jump, instead, you throw your head and use it to roll where you need to go. Your main goal is to try and escape the dungeon you've been stationed in, and make it out to the outside world.
